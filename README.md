## Project Features

- **Continuous Integration/Continuous Deployment (CI/CD):** Automated pipelines Using GitHub Vercel
- **Automated Testing:** Using Cypress Unit Testing
- **External Database Management:** Using MongoDB


## Web App Features

- Log into Account
- Create an Account
- Set User Settings:

    - Upload Profile Picture
    - Change Measurement Units
    - Dark mode/ Light mode
- Create Recipes

    - Add Recipe Title
    - Add Image
    - Add Measurements
    - Add Steps
- Browse Recipes
